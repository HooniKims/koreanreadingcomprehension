3. AI 코치 요청과 국외 이전 가능성

AI 코치 버튼을 누른 경우에만 현재 문단, 고른 문장 번호, 작성한 이유·질문·대화 내용·요약 초안이 서비스의 코치 기능으로 전송됩니다. 배포 설정에 따라 Upstage API 또는 OpenAI API가 사용될 수 있으며, OpenAI API를 사용하는 경우 입력 내용이 미국 등 국외의 처리 환경으로 전송될 수 있습니다. 이 전송은 선택 기능의 힌트 생성에만 쓰이며, AI 코치를 쓰지 않으면 전송하지 않습니다.

OpenAI API는 서버의 OPENAI_API_KEY 설정이 있는 경우에만 사용됩니다. API 사업자의 데이터 처리 방식과 보유 기간은 각 사업자의 최신 정책을 따릅니다. 현재 코드의 OpenAI 요청은 store: false로 설정되어 있지만, 안전·오남용 방지 목적의 제한적 로그 등은 제공사 정책에 따라 처리될 수 있습니다.
